## ğŸ› Troubleshooting

**Problem: Button not showing?**
- Make sure you're on a Roblox game page
- Try refreshing the page (F5)
- Check extension is enabled in settings

**Problem: Servers not changing color?**
- Refresh after joining a server

**Problem: Random button not joining?**
- Check you have internet connection
- Make sure the game has available servers

**Problem: Not tracking servers?**
- Check extension is enabled
- Open DevTools Console (F12) - should have no errors
- Try clearing browser cache
